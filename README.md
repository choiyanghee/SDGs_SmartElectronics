# 🌍 SDGs 번영(Prosperity) — 스마트전자과 기술 포트폴리오 앱

## 프로젝트 개요
UN 지속가능발전목표(SDGs)의 **번영(Prosperity)** 테마를 기반으로, 특성화고 스마트전자과 학생들이 핵심 기술역량을 기록·관리하고 지속가능한 발전에 이바지하는 인재로 성장하는 것을 지원하는 웹 앱입니다.

## ✅ 구현 완료 기능

### 1. 🏠 메인 히어로 페이지
- SDGs 번영(Goal 8, 9, 11) 주제 소개
- 스마트전자과 비전 & 꿈 표현
- 5가지 핵심 역량 카드 (회로설계/PCB, IoT/센서, 아두이노, 조립·유지보수, 프로그래밍)
- 파티클 애니메이션 배경

### 2. 💼 내 기술 포트폴리오
- **학생 로그인**: 이름으로 간편 입장 (최대 40명 이상 지원)
- **등록된 학생 목록**: 빠른 로그인용 칩 표시
- **포트폴리오 CRUD**: 프로젝트 추가/수정/삭제
- **사진 업로드**: 클라이언트 측 이미지 리사이징 (800×600, JPEG 70%)
- **역량 카테고리 필터링**: 회로설계, PCB제작, IoT, 센서기술, 마이크로컨트롤러, 전자제품조립, 프로그래밍
- **프로젝트 상세보기**: 모달로 사진/설명 확인

### 3. 🏢 실제 채용공고 링크
- 잡코리아 고졸채용 검색 바로가기
- 사람인 고졸채용 검색 바로가기
- 고용24 고졸채용 정보 바로가기
- NCS 국가직무능력표준 직무분석 바로가기
- 취업 준비 TIP 4가지

### 4. 📜 자격증 취득 체크하기
- 7개 자격증 체크리스트 (전자캐드기능사, 전자기능사, 임베디드기능사, 전기기능사, 프로그래밍기능사, 컴퓨터활용능력, ITQ)
- 이름 기반 개인별 자격증 관리
- 취득일 기록
- 진행률 통계 (취득 완료/도전 중/퍼센트)

## 📁 파일 구조
```
index.html          # 메인 SPA 페이지
css/style.css       # 전체 스타일시트
js/app.js           # 애플리케이션 로직
README.md           # 프로젝트 문서
```

## 🗄️ 데이터 모델

### students 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| name | text | 학생 이름 |
| created_at | datetime | 가입일 |

### portfolios 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| student_name | text | 학생 이름 |
| category | text | 역량 카테고리 |
| title | text | 프로젝트 제목 |
| description | rich_text | 진행과정 설명 |
| image_data | text | 업로드 사진 |
| created_at | datetime | 작성일 |

### certificates 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| student_name | text | 학생 이름 |
| cert_name | text | 자격증 이름 |
| obtained | bool | 취득 여부 |
| obtained_date | text | 취득일 |

## 🔗 진입 URI
- `/` 또는 `/index.html` — 메인 앱 (SPA)
- 내부 네비게이션: 홈 / 기술 포트폴리오 / 채용공고 / 자격증

- HTML5, CSS3, Vanilla JavaScript
